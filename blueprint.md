# 문화예술 컨설턴트 웹사이트 구축 프로젝트

## 1. 프로젝트 개요

이 프로젝트는 문화예술 전문 경영 컨설턴트를 위한 개인 웹사이트를 구축하는 것을 목표로 합니다. 사용자는 이 웹사이트를 통해 자신의 전문적인 인사이트를 블로그 형태로 공유하고, 문화예술 기획이나 독서모임과 같은 스터디 그룹을 모집하고 신청자를 받을 수 있습니다.

## 2. 프로젝트 상세 설계

### 가. 스타일 및 디자인

- **전체 컨셉:** 전문가의 신뢰성과 예술적 감성을 동시에 보여줄 수 있는 모던하고 세련된 디자인
- **색상 팔레트:** 메인 키 컬러 `#04C4C3`를 중심으로 화이트, 라이트 그레이 등을 사용하여 깨끗하고 활기찬 느낌을 줍니다.
- **폰트:** 가독성이 좋은 고딕 계열 폰트와, 제목 등 강조가 필요한 부분에는 세련된 세리프 폰트를 혼용
- **레이아웃:** CSS Grid와 Flexbox를 사용한 반응형 레이아웃으로, 모바일과 웹 환경 모두에서 최적의 경험을 제공
- **인터랙션:** 부드러운 애니메이션과 전환 효과를 적용하여 사용자 경험을 향상

### 나. 기능 및 컴포넌트

1.  **헤더 (Header)**
    *   웹사이트 로고
    *   네비게이션 메뉴 (소개, 인사이트, 스터디 모집)

2.  **소개 (About Me)**
    *   컨설턴트의 사진과 자기소개를 보여주는 섹션입니다.
    *   사진은 원형으로 표시하여 부드러운 느낌을 줍니다.

3.  **인사이트 섹션 (Blog)**
    *   블로그 게시물 목록을 보여주는 영역
    *   각 게시물은 `<blog-post-preview>` 웹 컴포넌트로 표시

4.  **스터디 모집 섹션 (Study Groups)**
    *   모집 중인 스터디 목록을 보여주는 영역
    *   각 스터디는 `<study-card>` 웹 컴포넌트로 표시

5.  **스터디 신청 폼 (Application Form)**
    *   '신청하기' 버튼 클릭 시 나타나는 모달 또는 별도 섹션
    *   `<application-form>` 웹 컴포넌트로 구현
    *   수집 항목: 이름, 나이, 성별, 휴대전화번호, 메일주소, 관심분야, 지향점

6.  **푸터 (Footer)**
    *   저작권 정보
    *   연락을 위한 이메일 주소

### 다. 기술 스택

- **Front-end:** 순수 HTML, CSS, JavaScript (Framework-less)
- **UI Components:** 웹 컴포넌트 (Web Components) 표준 기술 사용

## 3. 현재 진행 계획 (Phase 2)

1.  **`style.css`**: 키 컬러 변경 및 '소개' 섹션 스타일 추가
2.  **`index.html`**: '소개' 섹션 마크업 및 네비게이션 메뉴 업데이트, 푸터에 이메일 추가
3.  **`main.js`**: (필요시) 인터랙션 추가
